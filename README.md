is-pi
=====
Determines whether something is equal to PI or not.

## Installation

* Per project: `npm i is-pi --save`
* Globally: `npm i -g is-pi`

## Usage

```javascript
  var isPI = require('is-pi');
  console.log(isPI(3.14)); 		// true
  console.log(isPI('unicorn')); // false
```
